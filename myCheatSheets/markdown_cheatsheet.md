# MarkDown CheatSheet
- Link リンク

```
［表示文字］（URL）でリンクに変換
```

[ググる](https://www.google.co.jp/)

- 外部参照リンク
URLが長い/同じリンクを複数回利用する場合、リンク先への参照を定義可能
```
[ググる?][Google]
[Google]: https://brave.com/ja/
```

[ググる?][Google]
[Google]: https://brave.com/ja/
